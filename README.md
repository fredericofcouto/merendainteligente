# 🍽️ Merenda Inteligente

**Merenda Inteligente** é uma aplicação web desenvolvida com **Next.js**, **TypeScript**, **Tailwind CSS** e **Firebase**, projetada para modernizar e facilitar o gerenciamento de sistemas relacionados à alimentação escolar.

---

## 📁 Estrutura do Projeto

```bash
merendainteligente/
├── src/                 # Código-fonte da aplicação
├── docs/                # Documentação adicional
├── .vscode/             # Configurações do Visual Studio Code
├── public/              # Arquivos públicos
├── apphosting.yaml      # Configurações de hospedagem Firebase
├── package.json         # Dependências e scripts
├── tailwind.config.ts   # Configuração do Tailwind CSS
├── tsconfig.json        # Configuração do TypeScript
├── next.config.ts       # Configuração do Next.js
└── README.md            # Documentação do projeto

**🚀 Tecnologias Utilizadas**
Next.js – Framework React com suporte SSR e SSG

TypeScript – Tipagem estática em JavaScript

Tailwind CSS – Framework CSS utilitário

Firebase – Plataforma backend da Google

**  ⚙️ Como Executar o Projeto ** 
**  ✅ Pré-requisitos ** 
Node.js instalado (versão recomendada LTS)

Conta no Firebase (opcional, para hospedagem)

**  📥 Clonar o Repositório ** 

git clone https://github.com/fredericofcouto/merendainteligente.git
cd merendainteligente

**  📦 Instalar Dependências ** 

npm install

**  ▶️ Rodar em Ambiente de Desenvolvimento ** 

npm run dev

Acesse http://localhost:3000 no navegador.

**  ☁️ Deploy no Firebase ** 
**  🔧 Instalar CLI do Firebase ** 

npm install -g firebase-tools

**  ☁️ Deploy no Firebase ** 
**  🔧 Instalar CLI do Firebase ** 

npm install -g firebase-tools

**  🔐 Fazer Login ** 
firebase login

**  🏗️ Inicializar Projeto ** 

firebase init

**  🚀 Deploy da Aplicação ** 

firebase deploy
